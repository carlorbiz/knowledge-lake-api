## Troubleshooting Quick Fixes

### Can't find command_text
**Check**: Parse Slack Command output, find actual field name

### 401 from Manus API
**Fix**: Verify API_KEY header name (case-sensitive) and correct API key value

### Notion fails
**Fix**: Match property names exactly as they appear in your Notion database

### No response in Slack
**Fix**: Check response_url is being captured from Parse Slack Command node
