### 4. Medicare Provider Number (MPN) and Billing

| Checkpoint | Yes | No | Comments |
|------------|-----|----|----------|
| **Is there a current MPN for the registrar?** | ☐ | ☐ | |
| **Has the registrar submitted their MPN application at least 8 weeks prior to commencement?** *(Processing time: 4-6 weeks)* | ☐ | ☐ | |
| Is the MPN active and is the registrar billing Medicare? | ☐ | ☐ | |
| **Multi-site placements: Does the MPN cover all (accredited) additional sites?** | ☐ | ☐ | |

**Guidance Note to PM:**
- **Multi-site coverage:** Separate MPN applications must be submitted for each accredited location
- All branch and associated sites must have MPN coverage before registrar bills at those locations
- If MPN not yet active, registrar should not commence billing until MPN is issued by Medicare

### 5. Employment Contract and NTCER Compliance

| Checkpoint | Yes | No | Comments |
|------------|-----|----|----------|
| **Is the employment contract the current GPSA/GPRA template?** | ☐ | ☐ | |
| **Is the employment contract compliant with NTCER Version 2025-02 (effective 1 July 2025)?** | ☐ | ☐ | |

**Key Provisions to Verify:**
- ☐ 12% Superannuation Guarantee (from 1 July 2025, paid with salary from 1 July 2026)
- ☐ Monthly billing cycle (not fortnightly or weekly)
- ☐ Mandated teaching time quarantined (3 hours GPT1, 1.5 hours GPT2)
- ☐ Educational release time (16 days per 12 months)
- ☐ Leave entitlements documented (annual leave, personal/carer's leave, parental leave, study leave)

| **For AMS: Has a copy of the employment agreement been provided to the TPA before commencement?** *(Required for AMS Salary Support processing)* | ☐ | ☐ | |
