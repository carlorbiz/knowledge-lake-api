## ğŸ”§ Troubleshooting

### Problem: Template not creating documents
**Check:**
- Is template activated in DocsAutomator?
- Are all required placeholders filled?
- Is Drive folder accessible?
- Are permissions set correctly?

### Problem: Files created but URLs don't work
**Check:**
- Folder permissions: Should be "Anyone with link can view"
- Sharing settings on files themselves
- Try accessing in incognito browser

### Problem: Styling not applied
**Check:**
- Template design in DocsAutomator editor
- Markdown conversion settings
- Google Doc style inheritance

### Problem: Zapier MCP not working
**Check:**
- DocsAutomator action enabled in Zapier MCP config
- Claude Code restarted after adding action
- Credentials properly connected in Zapier
